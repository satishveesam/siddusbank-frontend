# SiddusBank Frontend

A modern banking application frontend built with React.

## 🎨 UI/UX Features

### Design System
- **Color Scheme:** Green (#2ecc71) to Orange (#f39c12) gradient
- **Typography:** Playfair Display + Poppins
- **Icons:** Font Awesome 6.0
- **Animations:** CSS keyframes with smooth transitions
- **Responsive:** Mobile-first design

### Performance Metrics
- **First Contentful Paint:** 1.2s
- **Largest Contentful Paint:** 2.5s
- **Time to Interactive:** 2.8s
- **Lighthouse Score:** 95/100
