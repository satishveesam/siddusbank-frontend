# SiddusBank Frontend 
 
A modern banking application frontend built with React. 
 
## Features 
- User authentication (login/register) 
- Dashboard with account statistics 
- Account management 
- Deposit, withdraw, and transfer money 
- Transaction history 
- Profile management 
- Responsive design 
 
## Technology Stack 
- React 18 
- React Router DOM 
- Axios 
- React Toastify 
- Chart.js 
- CSS3 
 
## Live Demo 
- Frontend: https://siddusbank-frontend.vercel.app 
- Backend API: https://siddusbank-backend-production.up.railway.app
